# 参考文献格式规范

## GB/T 7714-2015 （中国国家标准）

### 期刊论文 [J]
```
[序号] 作者. 题名[J]. 刊名, 出版年份, 卷号(期号): 起止页码.
```
示例：
```
[1] 张三, 李四. 人工智能在医疗诊断中的应用研究[J]. 计算机学报, 2023, 46(3): 521-538.
```

### 专著 [M]
```
[序号] 作者. 书名[M]. 出版地: 出版社, 出版年份.
```
示例：
```
[2] 王五. 机器学习导论[M]. 北京: 清华大学出版社, 2022.
```

### 学位论文 [D]
```
[序号] 作者. 题名[D]. 保存地: 保存单位, 年份.
```
示例：
```
[3] 赵六. 基于深度学习的图像识别研究[D]. 北京: 北京大学, 2024.
```

### 会议论文 [C]
```
[序号] 作者. 题名[C]//会议名称. 会议地点: 出版者, 年份: 起止页码.
```

### 英文期刊（GB/T 7714格式）
```
[序号] Author A A, Author B B. Title[J]. Journal Name, Year, Volume(Issue): Pages.
```
示例：
```
[4] Vaswani A, Shazeer N, Parmar N, et al. Attention is all you need[C]//Advances in Neural Information Processing Systems. Long Beach: NeurIPS, 2017: 5998-6008.
```

## APA 7th（美国心理学会）

### 期刊论文
```
Author, A. A., & Author, B. B. (Year). Title of article. Title of Periodical, Volume(Issue), page-range. https://doi.org/xxxx
```
示例：
```
Smith, J. D., & Jones, M. (2023). Social media effects on adolescent mental health. Journal of Adolescent Psychology, 45(2), 112-128. https://doi.org/10.xxxx/xxxxx
```

### 书籍
```
Author, A. A. (Year). Title of work: Capital letter also for subtitle. Publisher.
```

### 学位论文
```
Author, A. A. (Year). Title of dissertation [Doctoral dissertation/Master's thesis, Institution Name]. Database Name.
```

---

## MLA 9th（现代语言协会）

### 期刊论文
```
Author(s). "Title of Article." Title of Journal, Volume, Issue, Year, pages. Database, DOI.
```

---

## 格式选择建议

| 学科领域 | 推荐格式 |
|----------|----------|
| 理工科（计算机、工程、数学） | GB/T 7714 |
| 社会科学（经管、教育、心理） | APA 7th |
| 人文艺术（文学、语言学） | MLA 9th |
| 医学/生物 | AMA / 中华医学格式 |

**默认**：如果 paper_require 未指定引用格式，中文论文默认使用 **GB/T 7714**。
