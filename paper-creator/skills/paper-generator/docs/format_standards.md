# 论文排版格式标准

## 通用格式规范（本科/硕士毕业论文）

### 页面设置
| 属性 | 规范 |
|------|------|
| 纸张大小 | A4 (210mm × 297mm) |
| 页边距-上 | 2.54cm |
| 页边距-下 | 2.54cm |
| 页边距-左 | 3.18cm |
| 页边距-右 | 3.18cm |
| 装订线 | 0cm |

### 字体字号
| 元素 | 中文字体 | 英文字体 | 字号 | 加粗 |
|------|----------|----------|------|------|
| 论文标题 | 黑体 | Times New Roman | 小二(18pt) | 是 |
| 一级标题 | 黑体 | Times New Roman | 三号(16pt) | 是 |
| 二级标题 | 黑体 | Times New Roman | 四号(14pt) | 是 |
| 三级标题 | 黑体 | Times New Roman | 小四(12pt) | 是 |
| 正文 | 宋体 | Times New Roman | 小四(12pt) | 否 |
| 图表标题 | 宋体 | Times New Roman | 五号(10.5pt) | 是 |
| 页眉 | 宋体 | Times New Roman | 五号(10.5pt) | 否 |
| 页码 | Times New Roman | - | 五号(10.5pt) | 否 |
| 参考文献 | 宋体 | Times New Roman | 五号(10.5pt) | 否 |

### 段落格式
| 属性 | 规范 |
|------|------|
| 行距 | 1.5倍行距（固定值22磅） |
| 段前间距 | 0行 |
| 段后间距 | 0行 |
| 首行缩进 | 2字符 |
| 对齐方式 | 两端对齐 |

### 标题格式
- 一级标题：居中，段前段后各1行间距
- 二级标题：左对齐，段前段后各0.5行间距
- 三级标题：左对齐，段前段后各0.5行间距

### 图表编号
- 图编号：图X-Y（X=章号，Y=图序），标题置于图下方，居中
- 表编号：表X-Y（X=章号，Y=表序），标题置于表上方，居中

### 页眉页码
- 奇数页页眉：论文题目（居中）
- 偶数页页眉：章节标题（居中）
- 页码：页面底端居中，从正文第一页开始编号

### 摘要格式
- 中文摘要：标题"摘要"黑体三号居中，正文宋体小四
- 英文摘要：标题"Abstract" Times New Roman 三号居中加粗，正文 Times New Roman 小四
- 关键词：3-5个，分号分隔

### 目录
- 自动生成，含一至三级标题
- "目录"二字黑体三号居中
- 目录内容宋体小四，1.5倍行距

---

## Word文档兼容性要求

- 兼容 Microsoft Office 2016 / 2019 / 365
- 兼容 WPS Office 2019 及以上版本
- 字体嵌入文档（避免在其他电脑上字体丢失）
- 不使用宏和VBA脚本（兼容性考虑）
