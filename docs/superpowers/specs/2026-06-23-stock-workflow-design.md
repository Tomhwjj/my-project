# 选股工作流设计

> 2026-06-23 | 状态：设计完成，待实施

## 一、目标

构建一套 Claude Code 选股工作流，辅助个人 A 股投资决策。核心定位：**你出策略，我出数据**——工作流负责数据获取、筛选排序、报告生成，投资决策由用户自己做出。

## 二、投资风格

| 维度 | 选择 | 说明 |
|------|------|------|
| 主风格 | 中线波段 | 持股几周到几个月 |
| 辅助 | 短线择时 + 长线底仓 | 短线辅助入场点，长线拿少数好公司 |
| 选股路径 | 先看板块，再挑个股 | 找到资金流入的强势板块，再从板块里选龙头 |
| 核心指标 | 资金面 > 基本面 > 技术面 | 主力净流入权重最高，PE/ROE 次之，均线/MACD 辅助 |
| 输出模式 | 日报 + 按需钻取 | 每日固定速报 + 自由深度查询 |

## 三、架构

三层分离：

```
用户（Claude Code）
   │ /daily-report / "看电力板块"
   ▼
Skill 层（策略编排）
   │ 选股工作流.md — 自然语言 → 结构化流程
   │ · 板块扫描 → 选股 → 风险速览
   │ · 分析逻辑、打分规则、输出格式
   │ 调用 Python 脚本取数
   ▼
Python 数据层（取数引擎）
   │ fetcher.py — 东方财富 API 直连
   │ · sectors: 行业板块排行
   │ · stocks <代码>: 板块成分股
   │ · detail <代码>: 个股深度
   │ 输出 JSON
   ▼
东方财富公开 API
   push2.eastmoney.com（免费）
```

## 四、数据层

### 文件：`fetcher/fetcher.py`

三个子命令：

| 命令 | 参数 | 数据内容 |
|------|------|---------|
| `sectors` | 无 | 全部行业板块：涨幅/主力净流入/成交额/5日趋势 |
| `stocks` | 板块代码如 BK0428 | 板块成分股：现价/涨跌幅/PE/量比/换手/主力净流入/总市值/5日涨幅 |
| `detail` | 个股代码如 600900 | 行情快照+资金流向+财务指标+技术指标(MA/MACD/KDJ) |

### 数据来源

- 板块列表：`push2.eastmoney.com/api/qt/clist/get?fs=m:90+t3`
- 成分股：`push2.eastmoney.com/api/qt/clist/get?fs=b:BKxxxx`
- 个股：`push2.eastmoney.com/api/qt/stock/get?secid=1.xxxxxx`

### 输出格式

所有命令统一输出 JSON，stdout 打印，Skill 层直接解析。

### 依赖

`requests` — 仅一个外部依赖。

## 五、Skill 层

### 文件：`skill/STOCK.md`

### 触发条件

- `/daily-report` — 自动跑完整日报
- "看板块" / "筛股票" / "分析 <股票>" — 按需钻取

### 日报工作流（/daily-report）

**第 1 步 — 板块扫描**
1. 调 `python fetcher.py sectors`
2. 筛选「涨幅 > 0 且 主力净流入 > 0」的板块
3. 涨幅排名 + 资金排名取交集 → 锁定 3-5 个攻击方向
4. 检查板块 5 日趋势（连红/震荡/转弱）

**第 2 步 — 选股**
1. 对每个锁定板块，调 `python fetcher.py stocks <代码>`
2. 加权打分：
   - 资金面 (50%)：主力净流入占比、量比
   - 基本面 (35%)：PE、ROE、营收增速
   - 技术面 (15%)：均线形态、5日动量
3. 每板块取 TOP3，去重，生成标的池

**第 3 步 — 风险速览**
1. 调大盘资金流向、北向资金、涨跌家数
2. 综合判断：进攻 / 观望 / 防守

**第 4 步 — 输出**
结构化 Markdown，包含板块表 + 标的池表 + 风险提示。

### 钻取工作流（自由对话）

用户自然语言 → Skill 解析意图 → 调对应 Python 命令 → 分析 → 输出。

### 输出格式约定

```markdown
## 📊 YYYY-MM-DD 收盘速报

### 🔥 强势板块
| 排名 | 板块 | 涨幅% | 主力净流入 | 5日趋势 |

### 🎯 标的池
| 股票 | 板块 | 涨幅% | 主力净比 | PE | ROE | 评分 |

### ⚠️ 风险速览
大盘资金 / 北向 / 策略建议
```

## 八、约束

- 数据仅供分析参考，不构成投资建议
- 投资决策由用户自行做出

- 请求频率控制，避免触发限流
